# Reveal.Me

## Description

A full stack web dating application for discovering genuine connections ! Meet your match and slowly reveal your partner's pictures through chatting

Built with ReactJS and TailwindUI for frontend and NodeJS for backend

### Dependencies

Database:
- MongoDB

Backend :

- Node
- Express JS
- Nodemon
- CORS
- Body parser
- Bcrypt
- Socket.io
- Jsonwebtoken
- dotenv
- Mocha
- Chai
- Supertest

Frontend :

- Vite
- React
- React-dom
- React-router
- Axios
- TailwindUI
- DaisyUI
- Socket.io
- Vitest
- React-testing-library

## Getting Started
Scripts:

- Backend : npm start

- Frontend : npm run dev
